# Projet Covoiturage - Touche pas au klaxon !

## Installation
1. Importer le fichier SQL fourni dans votre serveur MySQL local.
2. Configurer les accès à la base de données dans le fichier `src/Database.php`.
3. Lancer la commande `composer install` dans le terminal pour installer le routeur et les dépendances.
